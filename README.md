# hangman
Hangman is a command-line game written in Python. 
The user has to guess the hidden word before running out of turns

This project is a simple digital version of the classic Hangman game. 
A word is selected and the user tries to guess the hidden word by either guessing a letter or part of the word.
The game keeps track of already used letters, remaining turns and shows the user's progress.

# Getting started
Before running this program make sure you have:
- Python 3.x
- Windows 10/11 (This project was only tested on Windows)
- A code editor (I used VScode)

# Installing
- Go to my release (after clicking on 'Try this project, you will go to my release)
- Click on 'hangman.py'
- Open the downloaded file in a code editor

# Execution program
- Open the terminal
- Navigate to the folder called 'Hangman'
- Run: 'py hangman.py'
